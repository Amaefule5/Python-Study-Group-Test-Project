# 🐍 Python Study Group Test Project

A modern, responsive, and visually engaging landing page created for the **Python Study Group with Coach Timothy**. This platform introduces aspiring data professionals to a structured 30-day Python learning journey focused on **Data Analytics and Data Science**.

The website provides information about the program structure, curriculum, rules, benefits, certification, and registration process through a clean and interactive user interface.

## Live Demo

🔗 View the live website: **https://python-study-group.netlify.app/**

---

## Project Overview

This project was designed to attract and onboard new members into the Python Study Group. It emphasizes:

- A professional and modern design
- Clear learning pathway presentation
- Mobile-friendly experience
- Smooth navigation and animations
- Strong call-to-action for registration

---

## Features

### Hero Section
- Engaging introduction to the Python Study Group
- Highlight badges for key program information
- Interactive Python-themed code card
- Clear "Start Learning" call-to-action

### Program Structure
- 20 structured lessons over 30 days
- Learning Management System (Google Classroom) information
- Community-based learning approach
- Progress and accountability system

### Rules & Policies
- Assessment requirements
- Review call attendance policy
- Case study submission guidelines
- Learning prerequisites

### Certification Section
- Certificate of Completion information
- Digital achievement badge showcase
- Certification processing details
- Re-entry policy information

### Learning Benefits
- Hands-on practical projects
- Real-world case studies
- Team collaborations
- Portfolio development
- Career guidance and mentorship

### Responsive Design
- Mobile navigation menu
- Flexible grid layouts
- Optimized experience across all screen sizes

### Interactive Experience
- Smooth scrolling navigation
- Hover effects
- Scroll-triggered animations using Intersection Observer
- Dynamic mobile menu interactions
